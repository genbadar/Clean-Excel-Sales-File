# Clean Excel Sales Data Files

This repository contains Excel files intended for practicing **cleaning messy spreadsheets**, fixing structural and data-quality issues, and preparing files to be **reporting-ready**.

These files are suitable for:
- Excel data cleaning practice
- Spreadsheet restructuring
- Reporting and analytics preparation
- Power Query / Power BI readiness checks

## Practice Files

- **Sample Sales Data.xlsx**

- Removing structural issues
